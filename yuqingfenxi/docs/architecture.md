┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│  前端展示层  │ ←→ │  后端服务层  │ ←→ │  AI模型层   │
└─────────────┘    └─────────────┘    └─────────────┘
       ↑                   ↑                   ↑
       │                   │                   │
┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│   用户交互   │    │  数据存储   │    │ 外部数据源  │
└─────────────┘    └─────────────┘    └─────────────┘
```

## 2. 核心数据流
1. 数据采集：后端从外部数据源获取原始舆情数据
2. 数据处理：后端清洗和预处理数据
3. 模型分析：调用AI模型进行情感分析、主题聚类等
4. 结果存储：分析结果存入数据库
5. 前端展示：前端通过API获取数据并可视化
